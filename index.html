Опыт работы и проекты

Тестовое задание, Самообучение

Летом 2024 года я начала самообучаться по доступным ресурсам (канал Владилена Минина, современный учебник JavaScript, статьи на Habr и т. д.). Откликаясь на вакансии и взаимодействуя с работающими специалистами, я получила тестовое задание, которое требовало работу с API и LocalStorage, реализацию многостраничного сайта и навигации. Я выполнила его в виде адаптивного многостраничного сайта с помощью React.

Фулстак-разработка (Сайт для продавца), Пет-проект

Небольшой пет-проект для продавца изделий лазерного станка. Проект реализован на React с использованием хуков useEffect и useState. Также в нем осуществлено взаимодействие с телеграм-ботом через запрос к PHP-файлу на бэкенде, который передает информацию из формы продавцу. Страница обладает адаптивным дизайном в минималистичном стиле. Хостингом и обслуживанием сайта также занимаюсь я.

Frontend-разработчик (верстальщик), Дизайн-студия

С июля 2025 года я создавала адаптивные и привлекательные рекламные лендинги, учитывая UX-требования. Использовала современные технологии HTML, CSS и JavaScript для реализации динамических и взаимодействующих интерфейсов. Необходимо было работать с формами и отправкой данных на сервера. Сотрудничала с командой дизайнеров и маркетологов для достижения эффективных рекламных решений. Работа студии была ориентирована на зарубежные рынки и требовала проверки вводимых пользователем данных с помощью библиотек.
